About Wayfarer

A planner that runs entirely in your browser

Wayfarer is a single-page, framework-free travel planner. Your Groq API key and every itinerary you generate stay in your browser's local storage — nothing is sent to a server we control. Bring your own key, plan freely, and take your data with you when you go.

01

No backend

HTML, CSS and vanilla JavaScript only — the Groq API is the only external call.

02

Local-first

Trips, history and your API key live in localStorage on this device.

Built for detail

Seventeen itinerary sections per trip, from packing lists to local phrases.
